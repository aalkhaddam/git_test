Lesson: Links

Links: use the a element - called Anchor

About Odin Project

To add a link/destination, we use an attribute within the element by doing: href="link"

To change where the link is opened, use the target element

The rel attribute describes the relation between the current page and the link

  • noopener value: prevents the opened link from gaining access to the webpage from which it was opened
  • noreferrer value: prevents the opened link from knowing which webpage or resource has a link or reference to it
    • has noopener behaviour so can be by itself
    • both of these present security for phishing attacks (noopener) because it stops the new webpage from changing the old webpage to a new one (tabnabbing)
    • new browsers have security measures for this, but it's still good to pair target blank with those 2 rel values

Absolute links are links to pages on websites on the internet

  • have the following parts: protocol://domain/path

Relative links are links to pages within our own website

  • do not contain domain name since it assumes the domain will be the same
  • only include file path
  • This is a relative link: About
  • if linking something within your directory, add ./ before it to not run into issues (e.g., ./pages/about or ./index)
  • to go up a directory, double .. - e.g., ../folder_in_parent_directory/etc
